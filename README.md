# ACME-SlurpLog

ACME-SlurpLog is a tool to read Apache2 log files. No more, no less.
It is not an industrial strength solution to all the world's problems.

# INSTALLATION

To install this module, run the following commands:

	perl Makefile.PL
	make
	make test
	make install
